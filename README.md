# 拼图游戏

这是一个基于 Web 的拼图游戏，支持自定义图片上传和多种难度选择。

## 功能特点

1.  **图片上传**：支持上传本地图片作为拼图素材。
2.  **难度选择**：
    - 简单 (3x3)
    - 一般 (5x5)
    - 困难 (8x8)
    - 地狱 (15x15)
3.  **真实拼图形状**：使用 Canvas 生成带有凹凸接口的不规则拼图块。
4.  **交互体验**：
    - 左侧为散乱拼图区，右侧为拼接区。
    - 支持鼠标拖拽。
    - 自动吸附功能：当拼图块接近正确位置时会自动吸附。
    - 完成提示。

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术栈

- HTML5
- CSS3 (Flexbox)
- JavaScript (Canvas API, Drag & Drop)
